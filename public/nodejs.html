About Node.js

Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside a web browser. Node.js lets developers use JavaScript to write command line tools and for server-side scripting.

Key Features

  • Asynchronous and Event-Driven
  • Single-Threaded but Highly Scalable
  • No Buffering
  • License - Released under the MIT license
// Example Node.js server
import http from 'http';

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World\n');
});

server.listen(3000, '127.0.0.1', () => {
  console.log('Server running at http://127.0.0.1:3000/');
});

Working with Files

Node.js provides powerful APIs for working with the file system: 

// Reading and writing files in Node.js
import fs from 'fs/promises';
import path from 'path';

async function readAndWriteFile() {
  try {
    // Read a file
    const content = await fs.readFile('input.txt', 'utf-8');
    
    // Process the content
    const processedContent = content.toUpperCase();
    
    // Write to a new file
    await fs.writeFile('output.txt', processedContent);
    
    console.log('File processing completed successfully');
  } catch (error) {
    console.error('Error processing file:', error);
  }
}

readAndWriteFile();

Node.js Modules

Node.js has a built-in module system that allows you to organize your code into reusable pieces: 

// Built-in modules
import fs from 'fs';
import path from 'path';
import http from 'http';

// Third-party modules (installed via npm)
import express from 'express';
import lodash from 'lodash';

// Local modules
import { add, subtract } from './math.js';
import User from './models/User.js';

NPM (Node Package Manager)

NPM is the default package manager for Node.js and is used to install, share, and manage dependencies in Node.js projects: 

# Initialize a new Node.js project
npm init -y

# Install a package
npm install express

# Install a development dependency
npm install --save-dev nodemon

# Run a script defined in package.json
npm run dev
