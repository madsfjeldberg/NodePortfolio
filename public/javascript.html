JavaScript Fundamentals

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It is a language that is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm.

Key Features

  • First-class functions
  • Prototype-based object orientation
  • Asynchronous programming with Promises and async/await
  • Closures
  • Event-driven programming
// Example JavaScript code
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));

Modern JavaScript Features

ES6 and beyond introduced many powerful features to JavaScript: 

// Arrow functions
const add = (a, b) => a + b;

// Destructuring
const { name, age } = person;
const [first, ...rest] = items;

// Template literals
const message = `Hello, ${name}! You are ${age} years old.`;

// Spread operator
const newArray = [...oldArray, newItem];
const newObject = { ...oldObject, newProp: value };

// Async/await
async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Error fetching data:', error);
  }
}

DOM Manipulation

JavaScript can be used to manipulate the Document Object Model (DOM), allowing for dynamic web pages: 

// Selecting elements
const element = document.getElementById('myElement');
const elements = document.querySelectorAll('.my-class');

// Modifying content
element.textContent = 'New text content';
element.innerHTML = 'HTML content';

// Changing styles
element.style.color = 'blue';
element.classList.add('active');
element.classList.remove('inactive');

// Event handling
element.addEventListener('click', function(event) {
  console.log('Element clicked!', event);
});

JavaScript Modules

Modern JavaScript uses modules to organize code into reusable pieces: 

// math.js - Exporting functions
export function add(a, b) {
  return a + b;
}

export function subtract(a, b) {
  return a - b;
}

// main.js - Importing functions
import { add, subtract } from './math.js';

console.log(add(5, 3));      // 8
console.log(subtract(5, 3)); // 2

// Default exports and imports
export default class User {
  constructor(name) {
    this.name = name;
  }
}

import User from './user.js';
const user = new User('John');
